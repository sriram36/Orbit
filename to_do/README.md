# Orbit - Daily Routine & To-Do

Orbit is a beautiful, habit-focused to-do list application built with Flutter. It helps you stay on track with your daily routines, offering features like streaks, categorization, and a clean, reorderable interface.

## Features

*   **Daily Reset:** Routines reset automatically, perfect for building habits.
*   **Streaks:** Track your consistency with a daily streak counter.
*   **Categorization:** Organize tasks into Work, Personal, Health, or custom categories.
*   **Reorderable Interface:** Drag and drop to prioritize your routines (Grid and List views).
*   **Beautiful UI:** Modern design with smooth animations and haptic feedback.
*   **Notifications:** Set reminders for specific routines.
*   **History:** View your completion history.

## Getting Started

1.  **Prerequisites:** Ensure you have Flutter installed (`flutter doctor`).
2.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```
